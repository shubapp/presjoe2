ShubApp chat
===========
[Shubapp presentations site demo](http://pres.shubapp.com)

Usage
-----
```
git clone https://github.com/shubapp/presjoe2.git
cd presjoe2
node install
cd server
node server.js
```

then you can visit http://localhost/ and see it yourself
